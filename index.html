Full-Stack Engineer | Making Dents in the Universe

Hello! My Name is Albert.

I'm a Full-Stack Engineer bent on creating practical and beautiful apps. I have a big appetite for gettings lots of data and displaying it to the world.

I graduated from General Assembly's (GA) WDI program, a 12-week web development bootcamp where we were immersed in web technologies (ruby, ruby on rails, javascript, etc.) to create fully functional web applications.

Prior to GA, I was an Electrical Engineer with the US Army, working on wearable and vehicular acoustic sensors. After watching The Social Network, I became hooked to creating web applications through code.
